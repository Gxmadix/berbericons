Adding berbericons to your computer

  1. Download berbericons

  2. Install the berbericons.ttf file: OSX | Windows

Adding berbericons to your project

  1. Download and copy the berbericons folder into your project. This example assumes you've copied it into the root directory.

  2. Link to the berbericons.css stylesheet in the <head> of your html page:

  <link rel="stylesheet" href="/berbericons/berbericons.css">
  1. Simply use an icon in your HTML page:
  <span class="berbericon berbericon-yazz"></span>

If you choose to move the font files from the font folder, be sure to update the links in berbericons.css.

Best practices

  • Berber look best in sizes that are multiples of 16px.

  • berbericons are not monospaced. This lets them work well next to type, but it means they wonâ€™t stack nicely by default. If you intend to stack berbericons, such as in navigation, you will want to add some CSS to make them the same width, and centered. For example:

  .navigation .berbericon {
    width: 16px;
    text-align: center;
  }
